Student Task Manager (Web App)
A modern and visually appealing task management web application designed specifically for students to organize, prioritize, and track their academic tasks efficiently.
✨ Features
🔐 User Authentication (Login System)
➕ Create tasks with:
Title
Course name
Deadline
Priority (High / Medium / Low)
Tags
📊 Dashboard with:
Total tasks
Completed tasks
Completion percentage
🔍 Search functionality
🏷️ Tag filtering
📂 Task status filtering (All / Active / Completed)
💾 Automatic saving using Local Storage
🎨 Modern dark UI with responsive design
🧠 Project Highlights
Designed a clean and modern UI/UX inspired by real-world productivity tools
Implemented task prioritization system with visual indicators
Built real-time filtering and search system for better usability
Developed a dynamic dashboard showing task statistics
Ensured data persistence without backend using browser storage
🛠️ Technologies Used
HTML5
CSS3 (Modern UI / Dark Theme)
JavaScript (Vanilla JS)
Local Storage API
