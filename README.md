# Student Task Manager (Web App)

A modern task management web application designed for students to organize, prioritize, and track their academic tasks efficiently.

---

## Features

- User authentication (login system)
- Create tasks with:
  - Title
  - Course name
  - Deadline
  - Priority (High / Medium / Low)
  - Tags
- Dashboard with:
  - Total tasks
  - Completed tasks
  - Completion percentage
- Search functionality
- Tag filtering
- Task status filtering (All / Active / Completed)
- Automatic saving using Local Storage
- Responsive dark-themed UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API
